# Java Portfolio

A collection of Java projects demonstrating proficiency across the full spectrum of the language and ecosystem — from core fundamentals to production-grade Spring Boot applications.

## What's Inside

| Category | Topics Covered |
|---|---|
| **Data Structures & Algorithms** | Lists, trees, graphs, sorting, searching, dynamic programming |
| **Design Patterns** | Creational, structural, and behavioral GoF patterns |
| **Spring Boot Applications** | REST APIs, authentication, persistence, microservices |
| **Console / CLI Tools** | Command-line utilities and interactive programs |

## Tech Stack

- **Language:** Java 21
- **Framework:** Spring Boot 3
- **Build:** Maven / Gradle
- **Testing:** JUnit 5, Mockito
- **Database:** PostgreSQL, H2 (in-memory for tests)
- **Other:** Docker, Spring Security, Spring Data JPA

## Structure

```
java-portfolio/
├── algorithms/          # DSA implementations
├── design-patterns/     # GoF pattern examples
├── spring-boot-apps/    # Full Spring Boot projects
└── cli-tools/           # Console applications
```

## Running a Project

Each subdirectory contains its own `README.md` with build and run instructions. In general:

```bash
# Maven
./mvnw spring-boot:run

# Gradle
./gradlew bootRun
```
